Seja bem vindo a aba de instruções de uso. Caso queira pular as instruções e ir para a próxima aba, aperte "5". Para avançar para uma próxima aba ou um próximo item do cardápio, aperte "5". Para pular um item do cardápio, aperte "7". Para repetir o áudio de um produto ou aba, aperte "8". Para colocar um item no carrinho, aperte "9". Para voltar a uma aba anterior ou ao um item do cardápio passado anteriormente, aperte "4". Para finalizar o seu pedido, aperte "6". Para chamar o garçom para informar o seu pedido, tirar alguma dúvida ou pedir a conta, aperte "1".
